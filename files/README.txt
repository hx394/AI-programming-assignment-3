To run the code under Linux system:

1. put front_end.java, davis_putnam.java, back_end.java, and original input files in the same directory.

2. type in "javac front_end.java" and press enter.

3. type in "java front_end <inputFileName>" and press enter. FrontEndOutput.txt is generated.

4. type in "javac davis_putnam.java" and press enter.

5. type in "java davis_putnam <inputFileName>" and press enter. DavisPutnamOutput.txt is generated.

6. type in "javac back_end.java" and press enter.

7. type in "java back_end <inputFileName>" and press enter. The final result BackEndOutput.txt is generated.
